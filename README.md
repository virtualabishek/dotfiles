# My Dotfiles

Wanted that sleek Omarchy-on-Niri experience but on Arch with Noctalia? Yeah, me too. So I built my own. You can do the same if you want — just clone, stow, and you're done.

Managed with [GNU Stow](https://www.gnu.org/software/stow/) so everything stays clean and easy to maintain.

---

## What's Inside

```
.
├── fish/          # Fish shell config, functions & completions
├── fontconfig/    # JetBrainsMono Nerd Font as default monospace
├── foot/          # Foot terminal emulator + Noctalia theme
├── gtk/           # GTK3 theme generated by Noctalia (Gruvbox)
├── niri/          # Niri window manager config (KDL)
├── noctalia/      # Noctalia compositor/shell config
├── nvim/          # Neovim config (LazyVim-based)
└── tmux/          # Tmux config with vi-mode keybinds
```

### Niri + Noctalia
The combo that makes it all work. [Niri](https://github.com/YaLTeR/niri) is a scrollable-tiling Wayland compositor — think Omarchy-style window management. [Noctalia](https://github.com/NoctaliaDE/noctalia) brings the COSMIC shell on top with a panel, launcher, control center, notifications, lock screen, wallpapers, and even screen recording. Theme is Tokyo-Night with Gruvbox accents bleeding into GTK, foot, and niri.

### Fish Shell
Aliases I actually use: `t` for tmux, `c` for opencode, `f` for fastfetch. Comes with Copilot and Grok completions baked in.

### Foot Terminal
Lightweight, fast, and themed with Noctalia's colors. JetBrainsMono Nerd Font at size 10.

### Tmux
Prefix is `Ctrl+Space`, vi copy mode, easy pane splitting. The `D` keybind (or `tdl` function) spins up a three-pane layout with nvim, opencode, and a terminal — my daily driver.

### GTK3
Gruvbox-dark generated by Noctalia. Matches the rest of the desktop.

### Neovim
LazyVim-based. Clean, fast, gets out of your way.

### Fontconfig
Makes JetBrainsMono Nerd Font the default monospace everywhere.

## Want the Same Setup?

### What You'll Need (Arch Linux)

```bash
sudo pacman -S niri foot fish tmux neovim gnu-stow fontconfig
```

Noctalia is on the AUR:

```bash
yay -S noctalia
```

### Install

```bash
git clone https://github.com/your-username/dotfiles.git ~/dotfiles
cd ~/dotfiles
```

Then stow whatever you want:

```bash
stow fish      # fish config
stow niri      # niri window manager
stow noctalia  # noctalia compositor
stow foot      # foot terminal
stow tmux      # tmux config
stow gtk       # gtk3 theme
stow fontconfig # font config
stow nvim      # neovim config
```

This symlinks everything into place. `~/.config/niri/` -> `~/dotfiles/niri/.config/niri/` and so on.

Changed your mind? Just unstow:

```bash
stow --delete niri
```

## Keybinds

### Niri (Mod = Super)
| Key | Action |
|---|---|
| `Mod + Return` | Open foot terminal |
| `Mod + Q` | Close window |
| `Mod + D` | Noctalia launcher |
| `Mod + B` | Zen Browser |
| `Mod + E` | Thunar file manager |
| `Mod + 1-9` | Switch workspace |
| `Mod + Shift + 1-9` | Move window to workspace |
| `Mod + H/J/K/L` | Focus window left/down/up/right |
| `Mod + Ctrl + H/J/K/L` | Move window |
| `Mod + F` | Maximize column |
| `Mod + Shift + F` | Fullscreen |
| `Mod + V` | Toggle floating |
| `Mod + O` | Toggle overview |
| `Mod + Shift + L` | Lock screen |
| `Mod + Shift + T` | Toggle Noctalia bar |
| `Print` | Screenshot region |
| `Ctrl + Alt + Del` | Quit niri |

### Tmux (Prefix = Ctrl+Space)
| Key | Action |
|---|---|
| `Prefix + h/v` | Split pane (horizontal/vertical) |
| `Prefix + x` | Kill pane |
| `Prefix + D` | 3-pane coding layout |
| `Alt + 1-9` | Switch window |
| `Ctrl+Alt + Arrow` | Navigate panes |
| `Ctrl+Alt+Shift + Arrow` | Resize pane |
| `Alt + Enter` | Split pane below |
| `Prefix + q` | Reload config |

## Adding Your Own Stuff

Each component is its own Stow package. Want to add something else?

```bash
mkdir -p ~/dotfiles/newapp/.config/newapp
cp -r ~/.config/newapp/* ~/dotfiles/newapp/.config/newapp/
stow newapp
```
